Hiding elements

Opacity property

Saves the space in the layout and it is accessible by keyboard navigation and screen readers. 

        
          .opacity--0 {
            opacity: 0;
          }

Visibility property

Saves the space in the layout but it is not accessible by keyboard navigation or screen readers. 

        
          .visibility--hidden {
              visibility: hidden;
          }

Display property

Removes the item from the layout and it is not accessible by keyboard navigation or screen readers. 

        
          .display--none {
              display: none;
          }

Visually hidden helper class

Helper class often used to hide content but still make it reachable by keyboard navigation and screen reader. 

        
          .visually-hidden { 
            border: 0;
            clip: rect(0 0 0 0);
            height: 1px;
            margin: -1px;
            overflow: hidden;
            padding: 0;
            position: absolute;
            width: 1px;
          }

Relative font sizing

Browsers allow users to set the default font-size, if relative measure units are used to implement the design, we will make sure it holds when the font is scaled.

em:

Relative to the specified font-size of the nearest ancestor. 

            
              /* Extended practice to facilitate calculation of the em value */

              html {
                font-size: 62.5%
              }

              p {
                font-size: 1.6em; // 16px
              }

              /* Recomended approach */

              html {
                font-size: 100%; /* User defined setting */
              }

              body {
                font-size: 1em; /* 16px */
              }

              ul {
                font-size: 0.75em; /* 12px */
              }

              li {
                font-size: 1em; /* Relative to the ul element 12px */
              }
rem (root em):
Relative to the root element of the document, <html> 
            
                html {
                  font-size: 100%; /* Default 16px browser setting */
                }

                body {
                  font-size: 1rem; /* 16px */
                }
  
                ul {
                  font-size: 0.75rem; /* 12px */
                }
  
                li {
                  font-size: 1rem; /* Relative to the root element 16px */
                }

To calculate the size of the font in pixels, just divide the desired size of the font by its context (nearest ancestor with declared font-size defined for em, root element for rem)

The computed properties inspector will give you the size in pixels.
